# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.6.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT open a public issue.**
2. Use [GitHub's private vulnerability reporting](https://github.com/knuthelge/ZVisionGenerator/security/advisories/new).
3. Include: description, reproduction steps, and impact assessment.
4. You will receive acknowledgment within 7 days.

## Scope

This policy covers the `zvisiongenerator` Python package. Third-party model weights, LoRA files, and generated content are outside scope.
